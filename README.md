## 🚀 Featured Projects

### [TensorPy](https://github.com/TensorCraft/TensorPy)
A minimal Python interpreter written in C, with its own runtime, REPL, module system, exceptions, and growing standard-library-style layer.

### [TensorUI](https://github.com/TensorCraft/TensorUI)
A pure C UI framework for embedded and cross-platform apps, featuring modern layouts, gesture support, a HAL layer, and demo apps.

### [Memora](https://github.com/TensorCraft/Memora)
A privacy-first semantic memory extension for intelligent form assistance, with local AI support, retrieval, and WebGPU-accelerated similarity search.

### [MinerFormer](https://github.com/TensorCraft/MinerFormer)
A custom Transformer architecture for multi-timescale K-line forecasting with optional LLM feature fusion.
